## 📝 PASSO 3: CRIAR OS CALENDÁRIOS NO GOOGLE CALENDAR

### Para cada calendário, você vai:

1. **Criar o calendário**
   * Nome: Exatamente como está neste guia
   * Descrição: Copie do arquivo de rituais do departamento

2. **Compartilhar com os assinantes**
   * Clique em "Configurações" → "Compartilhar com colegas"
   * Adicione cada pessoa com **permissão de "Ver todos os detalhes do evento"**
   * ⚠️ Não adicione como "editor" a menos que gerencie o calendário

3. **Criar os eventos**
   * Use os dados de **Frequência**, **Duração**, **Horário** e **Checklist** dos arquivos de rituais
   * Para eventos recorrentes, defina a recorrência (ex: toda segunda-feira)
   * Copie a agenda completa para a descrição do evento

4. **Convidados ad-hoc**
   * Alguns eventos convidam pessoas de outros calendários
   * Essas pessoas **não** são assinantes, apenas convidadas para aquele evento
   * Exemplo: CEO é convidado para "Review de Criativos" mas não é assinante do calendário Marketing

## 🎯 PASSO 4: VALIDAÇÃO

### Checklist de validação por pessoa:
* [ ] Cada pessoa assina no máximo 2 calendários (seu time + Geral/Liderança, se aplicável)
* [ ] Cada pessoa vê todos os eventos em que participa (via subscription ou convite)
* [ ] Líderes estão em Geral + Liderança + seu time
* [ ] Colaboradores estão em seu time + Geral
* [ ] E-mails validados (sem typos, pessoas ativas no Google Workspace)

---

## 🚨 ERROS COMUNS A EVITAR

❌ **NÃO FAÇA:**
* Assinante em 3+ calendários por pessoa
* Criar eventos sem recorrência quando deveriam ser recorrentes
* Convidar pessoas para eventos sem confirmar participação esperada
* Deixar CEL/telefones no campo e-mail
* Usar nomes informais (sempre nome completo)

✅ **FAÇA:**
* Teste com 2-3 rituais piloto primeiro
* Comunique a mudança para a equipe com antecedência
* Deixe os arquivos de rituais acessíveis para dúvidas
* Revise os horários para evitar conflitos
* Ajuste conforme feedback dos times
